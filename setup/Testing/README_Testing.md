##Testing Utilities:

For Universal payloads / firmware control targets:
Tools are located in UniversalPayloadUtilities/linux (fork of libuniversalcan)

SetupVCAN.sh - Creates a virtual can0 on devices without canbus
UniCanMon - debug utility for Universal canbus targets. Allows user to call slave functions over canbus.
DummyPayloadSim - Simulates several payload targets including: Excavator, Arm (Old), Arm (New), Science, Electrical, BMS, Gimbal Camera
